Visual Studio empowers you to complete the entire development cycle in one place.
For example, you can edit, debug, test, version control, and deploy to the cloud.
With the diversity of features and languages in Visual Studio,
you can grow from writing your first piece of code to developing in multiple project types

A console application, in the context of C#, is an application that takes input and
displays output at a command line console with access to three basic data streams:
standard input, standard output and standard error.
